Age-Appropriate Development Paths

Select an age segment to explore the tools, technologies, and projects your camper will build.

Junior Explorers: Building Analytical Foundations

For our youngest campers, we prioritize tangible learning, spatial reasoning, and introductory coding concepts using block visual tools.

  • Scratch Junior: Program simple animations and interactive digital greeting cards.
  • LEGO Education WeDo: Build smart motor gears and program simple reactions to light sensors.
  • Tinkering Lab: Build mini-racers driven by solar panels and rubber bands.

Junior Outcomes:

100% Visual Block Mastery
4+ Working Motor Assemblies

Intermediate Architects: Expanding Engineering Depth

Middle campers transition into logical scripting and build more complex structural robotic projects.

  • Scratch & Python Basics: Transition from block coding to writing raw syntax commands.
  • LEGO SPIKE Prime: Build multi-wheeled rovers that navigate custom logic grids.
  • 3D Designing & Printing: Model original keychains and structures in CAD, then watch them print.

Intermediate Outcomes:

100% Basic Python Scripting
5+ Functional CAD Models

Senior Innovators: Advanced Applied Engineering

Senior students build production-ready projects, using text-based scripts, microprocessors, and custom electronic circuits.

  • Robotics with Arduino: Assemble and solder real microprocessors, wire resistors, and code in C++.
  • Unity 3D Game Engineering: Develop complex interactive physics engines and test original builds.
  • Web Development & AI: Write raw HTML/CSS websites and train basic machine learning datasets.

Senior Outcomes:

100% Text-Based Coding (Python/C++)
1+ Advanced Electronic Prototype to Keep
